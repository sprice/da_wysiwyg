Requirements
------------
wysiwyg-6.x-2.2 with the following patch:
http://drupal.org/node/624018#comment-2902814

Note
----
The serial id of the WYSIWYG input format that has an attached wysiwyg profile
is hardcoded in da_wysiwyg.install, da_wysiwyg.info and
da_wysiwyg.features.wysiwyg.inc. It is currently set to '100'. Only change this
value is you know what your are doing.
